Spring-hateoas-forms

An extension of spring-hateoas library to support forms in different formats

View project on GitHub

Welcome Spring HATEOAS Forms.

This library implements a common API for creating HATEOAS output in different implementations, like HAL, HAL-Forms, Syren, XHTML, etc. Additionally its API allows to define FORMS that are consumable by HATEOAS clients so that the restrictions of the expected inputs are formally defined

Integrations

This library can be integrated transparently with traditional Spring HATEOAS projects and also with Spring Data REST projects (even though the complete implementation it is still a work in progress).
